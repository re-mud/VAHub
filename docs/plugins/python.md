# Python-плагин
При выполнении команд python-плагины используют функции из скриптов.


## Манифест
В `manifest.json` укажите тип:
```json
"type": "python"
```


## Команды
В поле `commands` из `manifest.json` под значениями ключей укажите имя скрипта и его функцию в формате `имя:функция`, пример: `script.py:foo`.

Скрипт должен принимать один аргумент:
```python
def foo(text):
	pass
```
В качестве аргумента будет передан остаток текста команды.

Функция может вернуть json-строку, пример:
```python
import json

def say(text):
	return json.dumps({
		"speak": text
	})
```
