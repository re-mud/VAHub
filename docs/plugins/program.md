# Program-плагин
Program-плагин при выполнении команд запускает исполняемые файлы, такие как `exe` или `bat`.


## Манифест
В `manifest.json` укажите тип:
```json
"type": "program"
```


## Команды
В поле `commands` файла `manifest.json` указывайте относительный путь к исполняемому файлу плагина. 

- Пример простой команды: `"открой браузер": "open_browser.bat"`

- Команда с аргументами: `"открой браузер": "browser.bat --open"`

- Если аргумент содержит пробелы или должен быть строкой, заключайте его в кавычки: `"найди картинки с котиками": "browser.bat \"картинки с котиками\""`

- Для передачи остатка команды используйте `{text}`:
	```json
	"найди": "browser.bat {text}"
	```
	Здесь при команде `"найди картинки с котиками"` вызов будет: `browser.bat "картинки с котиками"`
