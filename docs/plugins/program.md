# Program-плагин
Program-плагин при выполнении команд запускает исполняемые файлы, такие как `exe` или `bat`.


## Манифест
В `manifest.json` укажите тип:
```json
"type": "program"
```


## Команды
В поле `commands` файла `manifest.json` указывайте относительный путь к исполняемому файлу плагина. 

- Пример простой команды: `"открой браузер": "open_browser.bat"`

- Команда с аргументами: `"открой браузер": "browser.bat --open"`

- Если аргумент содержит пробелы или должен быть строкой, заключайте его в кавычки: `"картинки с котами": "browser.bat \"картинки с котами\""`

- Для передачи текста вызова команды используйте {text}, который автоматически заменится на введенную команду: `"картинки с котами|картинки с собаками": "browser.bat {text}"`  
Здесь при команде «картинки с собаками» вызов будет: `browser.bat "картинки с собаками"`
